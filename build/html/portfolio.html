ABOUT ME

"저를 소개합니다."

일상에서 자주 사용하는 웹페이지를 사용자를 생각해서 만드는 것을 보며 감명깊게 여겼고,
사용자의 편의를 고려하며 개성을 표현하는 웹페이지를 만들고 싶어졌습니다. 그래서 저는 웹퍼블리셔의 길을 선택하였습니다.
Html, css중급, vanila script, jquery에 포커스를 맞춰 구현하였습니다.
sass를 활용하여 중복되는 css 구문을 제거하여 보다 깔끔한 반응형 웹을 구현하였고,
Gulp를 활용한 html, scss, js 작업 파일을 관리하였으며 git, github를 통해 코드관리를 하였습니다.
